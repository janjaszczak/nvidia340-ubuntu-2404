# Contributing

- **Languages:** README and issues may be **English** or **Polish**.
- **First-time git repo (maintainers):** if you received a tarball without `.git`, run `./scripts/bootstrap-git.sh` once (`git` required).
- **Before a PR:** run `bash -n install-nvidia340.sh` and **[ShellCheck](https://www.shellcheck.net/)** on `install-nvidia340.sh` (CI runs ShellCheck on pull requests).
- **Commits:** Conventional Commits are welcome but not mandatory (`fix:`, `docs:`, `chore:`).
- **Scope:** keep changes focused; large refactors should be discussed in an issue first.
